# Predictive Urban Drainage Blockage and Flash Flood Warning Node

## Overview

An Arduino-based smart drainage monitoring system designed to detect rising water levels and potential drainage blockages. The system automatically responds to flood risks using sensors, actuators, and Bluetooth notifications.

## Features

* Real-time water level monitoring using an ultrasonic sensor
* IR-based obstacle detection
* Automatic bypass control using a servo motor
* Water pump activation during critical water levels
* Vibration motor for debris removal
* Bluetooth alerts for remote status monitoring
* LED indicators for Normal, Warning, and Critical conditions

## Components

* Arduino Uno
* HC-SR04 Ultrasonic Sensor
* IR Sensor
* HC-05 Bluetooth Module
* SG90 Servo Motor
* Water Pump
* Vibration Motor
* L298N Motor Driver
* LEDs and Resistors

## How It Works

1. Monitors water level continuously.
2. Indicates status through LEDs.
3. Opens a bypass channel when water rises.
4. Activates vibration motor and pump during critical conditions.
5. Sends real-time alerts via Bluetooth.

## Applications

* Urban drainage systems
* Flood-prone areas
* Water tank monitoring
* Smart water management systems
